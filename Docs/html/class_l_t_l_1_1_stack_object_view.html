Класс LTL::StackObjectView

Представляет ООП доступ к стеку Lua. После назначения индекса объекта на стеке с ним можно работать. Очистка стека должна выполняться вручную. Подробнее...

#include <StackObject.hpp>

Подробное описание

Представляет ООП доступ к стеку Lua. После назначения индекса объекта на стеке с ним можно работать. Очистка стека должна выполняться вручную.
