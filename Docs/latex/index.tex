Lua Template Library (\doxylink{namespace_l_t_l}{LTL} for short) is a lightweight C++ 17 library for binding Lua 5.\+4. This library pursuits performance with safety and flexibility of C++ templates. It is completely static which means functions and methods used in binding are getting inlined and optimized by compiler.

\doxylink{namespace_l_t_l}{LTL} is made of following components\+:


\begin{DoxyItemize}
\item {\bfseries{User\+Data and Classes}} -\/ Used for defining C++ defined structure/class in Lua. Supports Methods and Properties.
\item {\bfseries{Function Wrapper}} -\/ Used to statically define {\ttfamily lua\+\_\+\+CFunction} which involves static type matching.
\item {\bfseries{Ref\+Object}} -\/ Represents object defined in Lua.
\item {\bfseries{State}} and {\bfseries{CState}} -\/ provide C++ interface for {\ttfamily lua\+\_\+\+State}; State class has template parameter for custom memory allocator.
\item {\bfseries{Stack\+Object\+View}} and {\bfseries{Stack\+Object}} -\/ classes for interacting with values on stack.
